Paper: "Lyapunov Differential Hierarchy and Polynomial Lyapunov Functions for Switched Linear Systems"

Authors: Matthew Abate, Corbin Klett, Samuel Coogan, and Eric Feron

Publisher: 2020 American Control Conference

Description: This script generates Figure 1 in the paper. Meta-Lyapunov functions of orders 2, 16, 20 and 26 are computed for a stable swicthed linear system and then plotted.

Code Author: Matthew Abate

Contact: Matt.Abate@GaTech.edu

Date: 2/17/2020

Notes: This code requires cvx, as well as ReachbleSet.mat which is provided.  Specifically, ReachableSet.mat contains the boundary of the reachable set for the system when beginning from x0 = [1 0].
